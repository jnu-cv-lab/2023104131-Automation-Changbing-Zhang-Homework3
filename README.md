# 图像下采样、恢复与频域分析实验

## 项目简介

本项目使用 OpenCV 和 Python 实现了图像的下采样（直接下采样与高斯平滑后下采样）、三种插值方法恢复（最近邻、双线性、双三次），并通过 MSE、PSNR、傅里叶频谱分析和分块 DCT 能量统计对结果进行客观与主观评价。实验基于经典的 Lena 灰度图像（512×512），完整覆盖了数字图像处理中采样、混叠、插值和频域分析的核心知识点。

## 环境要求

- Python 3.8+
- OpenCV 4.x
- NumPy
- Matplotlib

## 安装依赖

```bash
pip install opencv-python numpy matplotlib
```

## 快速开始

1. 确保项目目录下有一张测试图像（推荐 `lena.jpg`）。  
   如果没有，程序会自动从 OpenCV 官方 GitHub 仓库下载，或生成一个棋盘格图像作为替代。

2. 运行主程序


3. 程序将依次显示：
   - 空间域对比图（原图、直接缩小图、三种恢复图、高斯平滑缩小图）
   - 傅里叶频谱图（六幅）
   - 分块 DCT 系数图（四幅）
   - 终端输出 MSE、PSNR 及 DCT 低频能量占比

所有结果图片自动保存为：
- `spatial_comparison.png`
- `fft_spectra.png`
- `dct_coeffs.png`

### 主要函数说明

| 函数名 | 功能 |
|--------|------|
| `find_opencv_sample_image` | 自动查找 OpenCV 自带示例图片 |
| `mse` / `psnr` | 计算均方误差和峰值信噪比 |
| `show_spectrum` | 计算并显示对数幅度谱（中心化） |
| `block_dct` | 对图像进行 8×8 分块 DCT |
| `dct_low_energy_ratio` | 计算每个块左上角低频区域能量占比 |

## 实验结果摘要

### 空间域质量（PSNR）

| 恢复方法 | PSNR (dB) |
|----------|-----------|
| 最近邻   | 27.62     |
| 双线性   | 29.79     |
| 双三次   | 30.08     |

双三次插值恢复质量最佳，最近邻最差。

### DCT 低频能量占比（块内 4×4）

| 图像类型         | 低频能量占比 |
|----------------|--------------|
| 原图            | 0.854        |
| 最近邻恢复      | 0.912        |
| 双线性恢复      | 0.891        |
| 双三次恢复      | 0.879        |

恢复图像的低频能量占比均高于原图，表明插值引入了平滑效应。

### 频域分析

- 直接下采样图频谱出现混叠条纹（沿坐标轴方向的高频能量）。
- 高斯平滑后下采样图频谱干净，高频明显衰减。
- 最近邻恢复图的频谱存在规则网格状噪声（对应块状伪影）。
- 双三次恢复图的频谱最接近原图。
